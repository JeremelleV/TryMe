TryMe is a Chrome extension that adds an AI virtual try-on panel directly on shopping sites.  
Users can paste, upload, or capture garment and selfie images and send them to an IDM-VTON
backend on Hugging Face for AI-generated try-on results.

## Tech Stack

- Chrome Extension (Manifest V3)
- JavaScript, HTML, CSS
- IDM-VTON (`yisol/IDM-VTON`) via Hugging Face API
